# SMART-MAZE-SOLVER
Interactive Smart Maze Solver built with Python and Pygame. It uses the A*Algorithm to find the shortest path between two points while avoiding user-defined obstacles. Features include real-time search visualization and a grid reset function
Smart Maze Solver (A*)



*Quick Start
1. Requirements

Python 3.x

*Pygame library

2. Installation

Bash
pip install pygame
3. Run the App

*Bash
python maze_solver.py


 *Controls
Left Click: Place walls/obstacles.

Spacebar: Start the A* pathfinding.

R Key: Reset the grid.

*Core Logic (Module 2)
This project implements Informed Search using the A* algorithm:

Algorithm: f(n)=g(n)+h(n)

Heuristic (h): Manhattan Distance

Data Structure: Priority Queue (heapq) for optimal node selection.
